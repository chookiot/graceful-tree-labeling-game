Graceful Tree Labeling Game

The object of the game is to make a graceful labeled tree. Start by selecting N, the number of nodes in the tree. Click "Start a New Game" to generate a random tree with N nodes where the nodes are labeled 1,2,3,...,N. Select two nodes to swap their labels. Each edge label is the difference between its two nodes’ labels. Continue to swap node labels until the edge labels are exactly 1,2,3,...,N-1 (a graceful labeling). You can pause, resume, or start new games. The timer and move count persist through pauses. You can also switch between Radial and Hierarchical layouts.
